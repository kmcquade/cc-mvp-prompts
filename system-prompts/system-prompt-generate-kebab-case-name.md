# System Prompt: generate-kebab-case-name


## Summary

Instructs generating a short lowercase kebab-case conversation name in JSON.

# Raw Prompt Text
Generate a short kebab-case name (${NUM}-${NUM} words) that captures the main topic of this conversation. Use lowercase words separated by hyphens. Examples: "fix-login-bug", "add-auth-feature", "refactor-api-client", "debug-test-failures". Return JSON with a "name" field.
