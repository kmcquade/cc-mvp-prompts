# System Prompt: team-task-mailbox-operations

- Source: inline

## Summary

Instructions for spawning teammates, assigning tasks, and reading mailbox messages.

# Raw Prompt Text
Operation: spawn a teammate, spawnTeam to create a team, cleanup to remove team and task directories, read${PATH} mailbox messages, assignTask to assign a task to a teammate, or claimTask to self-claim an available task. Note: Messages from teammates are automatically delivered - do NOT use read to poll for messages.
